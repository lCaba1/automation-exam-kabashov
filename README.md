Приложение курьерской службы

Базовые функции
Получение информации о тарифах на доставку
Получение информации о времени прибытия курьера (Требует аутентификации)

Дополнение к функциям
При получении информации о времени прибытия курьера, возвращается также ФИО курьера и номер его
автомобиля

Аутентификация
Аутентификация происходит путем сверки присылаемого пользователем значения номера телефона и ID
доставки с имеющимися в БД. Если присланная пара номер-ID отсутствует в БД, возвращается сообщение об
ошибке.
